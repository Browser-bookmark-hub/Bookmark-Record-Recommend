## 使用规则

- 需要理解当前行为时，先看 `Bookmark-Record-Recommend-main/` 的代码和 `docs/PROJECT_STRUCTURE.md`。
- 文档标题中的状态优先于正文旧描述；正文里出现的“当前”“最终”“必须”“已落地”等字样可能已经过时。
- 推荐算法、AI 同步、浏览历史库、favicon、快捷复习和小组件都已经继续演进，必须以 `background.js`、`history_html/history.js`、`active_time_tracker/index.js` 和日历模块当前代码为准。
- 本目录不保存新的实施计划；新计划应放到活跃文档区，完成或废弃后再归档。

## 当前代码事实摘要

- `manifest.json` 当前版本为 `0.3.9`，入口为 Side Panel `panel-shell.html`，后台为 module service worker `background.js`。
- 当前快捷键命令包含 `_execute_action`、`open_recommend_view`、`open_additions_view`、`quick_review_next`。
- `background.js` 中当前推荐算法版本为 `SCORE_ALGO_VERSION = 6.4`。
- 浏览历史后台同步 alarm 为 `bb_history_background_sync_v1`，周期每日；失败重试 alarm 为 `bb_history_background_sync_retry_v1`，延迟 15 分钟。
- `active_time_tracker/index.js` 维护活跃会话和统计同步。
- `history_html/history.js` 承载 GitHub 同步、Markdown、快捷复习设置、widgets、recommend 和 additions 等大量 UI 逻辑。
